HTML CSS Javascript
HyperText Markup Language (HTML) is a markup language used to create webpages and web applications. HTML code defines the structure of a web page. Cascading Style Sheets (CSS) is a stylesheet language used to define the visual style of a webpage. A CSS file is usually linked in the HTML document. Javascript is a programming language used alongside HTML and CSS. It is used to make webpages more interactive, enabling video games to be played on a browser. Code snippets are applied inside <script> tags.
